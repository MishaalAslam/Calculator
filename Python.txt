Arrays
Basic syntax:
	array = [22,22,24,25,67,7]

for inserting any element in array we use syntax:
	array.insert(indexToBeTaken,value)

for automatically inserting any element at the end of array array we use syntax:
	array.append(value)

for eliminating any element from array we use syntax:
	array.remove(value)

for eliminating last element from array we use syntax:
	array.pop()					//print the last value and removes it from array

for eliminating any element from array by pop we use syntax:
	array.pop(indexNmbr)

for deleting whole array we use syntax:
	array.clear()
	
for targeting specific array element use syntax:
	print(aray[2])
	
for printing in a range of indexes we use syntax:
	print(array[2:5])			//output = (22,24,25)
	
for length of string/array we use len function
	len(array)
